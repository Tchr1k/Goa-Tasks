CSS-рЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў

CSS-рЃўрЃА рЃАрЃљрЃЏрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃў:

1. Inline CSS
рЃАрЃбрЃўрЃџрЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ, style рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ.
рЃЏрЃљрЃњ: <p style="color:red; font-size:20px;">рЃбрЃћрЃЦрЃАрЃбрЃў</p>
рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ: рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃгрЃарЃљрЃцрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃюрЃљрЃЎрЃџрЃў: рЃфрЃБрЃЊрЃљрЃЊ рЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ, рЃФрЃюрЃћрЃџрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў

2. Internal CSS
рЃДрЃЋрЃћрЃџрЃљ рЃАрЃбрЃўрЃџрЃў <head>-рЃерЃў <style> рЃбрЃћрЃњрЃерЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ.
рЃЏрЃљрЃњ: рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў
рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ: рЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃў рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў, рЃЎрЃљрЃарЃњрЃўрЃљ рЃЏрЃфрЃўрЃарЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃюрЃљрЃЎрЃџрЃў: рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ, рЃљрЃа рЃўрЃЦрЃћрЃерЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃќрЃћ

3. External CSS
рЃфрЃљрЃџрЃЎрЃћ рЃцрЃљрЃўрЃџрЃў (рЃЏрЃљрЃњ. styles.css) рЃЊрЃљ рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ <link> рЃбрЃћрЃњрЃўрЃЌ.
рЃЏрЃљрЃњ: <link rel="stylesheet" href="styles.css">
рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў! рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃЦрЃћрЃерЃўрЃарЃЊрЃћрЃЉрЃљ, рЃћрЃарЃЌрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА

рЃарЃЮрЃЏрЃћрЃџрЃў рЃљрЃЦрЃЋрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ?
External CSS Рєњ 90%+ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃћрЃАрЃљрЃљ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃў, рЃарЃљрЃЊрЃњрЃљрЃю:
Рђб рЃЎрЃЮрЃЊрЃў рЃАрЃБрЃцрЃЌрЃљрЃљ
Рђб рЃљрЃЊрЃЋрЃўрЃџрЃўрЃљ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃћрЃЉрЃљ
Рђб рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃЦрЃћрЃерЃљрЃЋрЃА Рєњ рЃњрЃЋрЃћрЃарЃЊрЃў рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ
Рђб рЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃЌ рЃЏрЃЌрЃћрЃџрЃў рЃАрЃљрЃўрЃбрЃў рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ

ID vs Class рЃАрЃћрЃџрЃћрЃЦрЃбрЃЮрЃарЃћрЃЉрЃў

ID рЃАрЃћрЃџрЃћрЃЦрЃбрЃЮрЃарЃў (#)
Рђб рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃўрЃљ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ (рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў id-рЃўрЃЌ)
Рђб рЃўрЃгрЃћрЃарЃћрЃЉрЃљ: #main-header { ... }
Рђб HTML-рЃерЃў: <h1 id="main-header">
Рђб рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃў рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃБрЃарЃЮрЃЉрЃљ (specificity) Рєњ рЃБрЃцрЃарЃЮ рЃФрЃџрЃўрЃћрЃарЃўрЃљ

Class рЃАрЃћрЃџрЃћрЃЦрЃбрЃЮрЃарЃў (.)
Рђб рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃДрЃЮрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ
Рђб рЃўрЃгрЃћрЃарЃћрЃЉрЃљ: .btn { ... }
Рђб HTML-рЃерЃў: <button class="btn">
Рђб рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљрЃЊрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў, рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.)

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ:
ID = рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃў (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў)
Class = рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљрЃЊрЃў рЃАрЃбрЃўрЃџрЃў (рЃЉрЃћрЃЋрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ)

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 3: 6 рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў

рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў Рђћ рЃерЃљрЃЋрЃў рЃцрЃЮрЃюрЃў + рЃЌрЃћрЃЌрЃарЃў рЃбрЃћрЃЦрЃАрЃбрЃў

рЃЏрЃћрЃЮрЃарЃћ рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў Рђћ рЃўрЃњрЃўрЃЋрЃћ рЃАрЃбрЃўрЃџрЃў

рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў Рђћ рЃўрЃњрЃўрЃЋрЃћ

рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў Рђћ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў рЃЌрЃћрЃЌрЃарЃў рЃцрЃЮрЃюрЃў

рЃЏрЃћрЃ«рЃБрЃЌрЃћ рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў

рЃЏрЃћрЃћрЃЦрЃЋрЃАрЃћ рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 4: рЃдрЃўрЃџрЃљрЃЎрЃў (рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ +10 рЃљрЃБрЃарЃљ рЃЏрЃЮрЃЋрЃўрЃњрЃЮ ­Ъўё)
